# Handmade™ — Branding Guideline: Design Philosophy

> Everything you need. In one cozy home.

Handmade's design identity is built around a **calming, beautiful experience** — craft-inspired colours, soft shapes, and delightful moments throughout the UI. The app should feel like picking up your favourite project on a quiet evening, not like navigating enterprise software.

---

## Key Principles

### Craft-Inspired Palette
Warm, tactile colours that echo yarn, wool, and natural materials. The UI should feel handmade and inviting, not sterile or techy.

### Soft Shapes
Rounded corners, gentle transitions, and approachable typography. Every element should feel touchable and friendly.

### Delightful Moments
Subtle animations and character illustrations (e.g., the yarn ball mascot) that bring personality without cluttering the experience. These micro-interactions reward engagement and reinforce the brand's warmth.

### Calm Over Feature-Dense
The interface should breathe — prioritise clarity and white space over cramming in controls. The goal is zero cognitive overhead so the user can stay in their crafting flow.
